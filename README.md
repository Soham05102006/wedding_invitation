# Vaibhav & Sneha - Wedding Invitation 💍

A high-end, mobile-responsive, and interactive digital wedding invitation designed with a traditional Indian Palace theme. This project features celestial animations, interactive "scratch to reveal" elements, and a dedicated memory gallery.

## ✨ Features

- **Royal Aesthetic**: Traditional Indian Palace theme with gold accents, deep teal backgrounds, and ornate typography.
- **Interactive Scratch Card**: A custom "Scratch to Reveal" card to unveil the wedding date.
- **Memory Gallery**: An automatically changing slideshow of cherished memories with premium ornate borders.
- **Live Countdown**: A real-time countdown timer to the wedding ceremony on May 14, 2026.
- **Program Timeline**: A detailed schedule of events from Mehendi to the Wedding Ceremony.
- **Interactive Map**: Integrated Google Maps for the home location and the Kanha Celebration Lawn.
- **Celestial Backdrop**: Dynamic star particles and floating lanterns for a magical atmosphere.
- **Background Music**: Integrated invitation song with smooth fade-in and user controls.

## 🛠️ Technology Stack

- **Framework**: React.js
- **Styling**: Tailwind CSS / Vanilla CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Asset Management**: Local optimized images and Unsplash integration

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Soham05102006/wedding_invitation.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## 📸 Assets

The `src/assets` folder contains all traditional motifs, lantern icons, and personal photos used across the application.
